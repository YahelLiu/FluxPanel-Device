# ESP32-C3 硬件接线指南

本文档详细说明 ESP32-C3 BLDC 电机控制系统的所有硬件接线方法。

---

## 📋 硬件清单

| 组件 | 型号 | 数量 | 说明 |
|------|------|------|------|
| MCU | ESP32-C3-DevKitM-1 | 1 | 4MB Flash, 160MHz, WiFi 内置 |
| 显示屏 | ST7735S TFT LCD | 1 | 1.8" 128x160 像素，SPI 接口 |
| 继电器 | 光耦隔离继电器模块 | 1 | 5V，低电平触发 |
| ESC | BLDC 电调 | 1 | 50Hz PWM 控制 |
| 电流传感器 | INA219 | 1 | I2C 接口，32V/2A 量程 |
| 霍尔传感器 | 霍尔开关模块 | 1 | 转速脉冲测量 |
| 电机 | BLDC 无刷电机 | 1 | 配合 ESC 使用 |

## 1️⃣ ST7735S TFT LCD 显示屏

### LCD 引脚顺序（从左到右）

```
┌───────────────────────────────────────┐
│ BLK  CS  DC  RST  SDA  SCL  VDD  GND │
└───────────────────────────────────────┘
```

### 接线表

| LCD 引脚 | ESP32-C3 GPIO | 说明 |
|----------|---------------|------|
| BLK | 不接 或 3.3V | 背光（可选） |
| CS | GPIO 7 | 片选信号 |
| DC | GPIO 6 | 数据/命令选择 |
| RST | GPIO 10 | 复位信号 |
| SDA | GPIO 3 | SPI 数据线 (MOSI) |
| SCL | GPIO 2 | SPI 时钟线 (SCK) |
| VDD | **5V** | 电源（推荐 5V） |
| GND | GND | 地线 |

### 接线图

```
    ST7735S LCD              ESP32-C3
   ┌─────────┐              ┌─────────┐
   │  BLK    │──────────────│  不接   │
   │  CS     │──────────────│  GPIO7  │
   │  DC     │──────────────│  GPIO6  │
   │  RST    │──────────────│  GPIO10 │
   │  SDA    │──────────────│  GPIO3  │
   │  SCL    │──────────────│  GPIO2  │
   │  VDD    │──────────────│  5V     │  ← 推荐 5V 供电
   │  GND    │──────────────│  GND    │
   └─────────┘              └─────────┘
```

### ⚠️ 供电注意事项

ESP32-C3 的 3.3V 引脚电流限制约 50mA，LCD + 背光可能超过此限制。

| 方案 | 接法 | 适用场景 |
|------|------|----------|
| **A: 5V 供电（推荐）** | VDD → 5V | LCD 支持 5V 输入 |
| B: 3.3V + 限流电阻 | VDD → 3.3V, BL → 150Ω → 3.3V | 兼容性好，背光暗 |
| C: 3.3V 无背光 | VDD → 3.3V, BL 不接 | 电流最小 |

## 2️⃣ 光耦隔离继电器

### 继电器模块引脚

```
┌─────────────────────────────────────┐
│  控制端（低压侧）                    │
│  DC+    DC-    IN                   │
│   ●      ●      ●                   │
│                                      │
│  ┌─────────────────┐                │
│  │   光耦隔离      │                │
│  └─────────────────┘                │
│                                      │
│  输出端（高压侧）                    │
│  COM    NO    NC                    │
│   ●      ●      ●                   │
└─────────────────────────────────────┘
```

### 控制端接线

| 引脚 | ESP32-C3 | 说明 |
|------|----------|------|
| DC+ | 5V | 模块供电正极 |
| DC- | GND | 模块供电负极（共地） |
| IN | GPIO 8 | 控制信号输入 |

### 输出端接线（接负载）

| 引脚 | 说明 |
|------|------|
| COM | 公共端 |
| NO | 常开触点 - 正常断开，触发时接通 COM |
| NC | 常闭触点 - 正常接通，触发时断开 COM |

### 接线图

```
ESP32-C3                光耦继电器模块
┌──────────┐           ┌──────────────────┐
│          │           │                  │
│    5V ───┼───────────┼── DC+            │
│          │           │                  │
│   GND ───┼───────────┼── DC-            │
│          │           │                  │
│  GPIO8 ──┼───────────┼── IN             │
│          │           │                  │
└──────────┘           │      ┌─────┐     │
                       │  COM │     │ NO  │
                       │   ●──┤     ├──●  │
                       │       └─────┘    │
                       └──────────────────┘
                              │       │
                              │       │
                         ┌────┴────┐  │
                         │  负载   │  │
                         │ (ESC等) │  │
                         └────┬────┘  │
                              │       │
                         电源负 ──────┘
```

> **说明**：
> - **DC+/DC-**：继电器模块本身的供电（5V 和 GND）
> - **IN**：控制信号，高/低电平控制继电器吸合
> - **COM/NO/NC**：输出触点，用于控制负载电路

### Fail-safe 设计

本项目使用**低电平触发**（`RELAY_ACTIVE_LOW = true`）：

```
ESP32 启动/重启时:
┌─────────────────────────────────────────┐
│ relay_init() 执行                        │
│ ↓                                       │
│ digitalWrite(RELAY_PIN, HIGH)           │
│ ↓                                       │
│ 继电器输入 = HIGH                        │
│ ↓                                       │
│ 因为低电平触发，高电平 = 继电器断开      │
│ ↓                                       │
│ 电源断开 → 安全状态 ✅                   │
└─────────────────────────────────────────┘
```

## 3️⃣ ESC PWM 控制

### 接线表

| ESC 引脚 | ESP32-C3 GPIO | 说明 |
|----------|---------------|------|
| 信号线 | GPIO 1 | PWM 50Hz 控制信号 |
| 电源正 | 继电器 NO | 受继电器控制 |
| 电源负 | GND | 地线 |

### PWM 信号规格

| 参数 | 值 | 说明 |
|------|-----|------|
| 频率 | 50 Hz | 标准 ESC 信号 |
| 分辨率 | 16 bit | LEDC 定时器 |
| 最小脉宽 | 1000 µs | 停止/最低转速 |
| 最大脉宽 | 2000 µs | 最高转速 |
| 中位脉宽 | 1500 µs | 中速 |

### 接线图

```
ESP32-C3                    ESC
┌──────────┐           ┌──────────┐
│          │           │          │
│  GPIO1 ──┼───────────┼── 信号   │
│          │           │          │
│   GND ───┼───────────┼── GND    │
│          │           │          │
└──────────┘           │  电源+   │←── 继电器 NO
                       │  电源-   │←── GND
                       └──────────┘
                            │
                       ┌────┴────┐
                       │  BLDC   │
                       │  电机   │
                       └─────────┘
```

### ESC 校准流程

```
1. 上电前：油门拉到最大
2. ESC 上电：听到 "滴滴" 两声
3. 油门拉到最小：听到 "滴" 一声
4. 校准完成，可以正常控制
```

## 5️⃣ 霍尔传感器

### 接线表

| 霍尔引脚 | ESP32-C3 GPIO | 说明 |
|----------|---------------|------|
| 信号 | GPIO 9 | 转速脉冲输入（中断） |
| VCC | 3.3V | 电源 |
| GND | GND | 地线 |

### 接线图

```
ESP32-C3                霍尔传感器模块
┌──────────┐           ┌──────────────┐
│          │           │              │
│  3.3V ───┼───────────┼── VCC        │
│          │           │              │
│   GND ───┼───────────┼── GND        │
│          │           │              │
│  GPIO9 ──┼───────────┼── OUT        │
│          │           │              │
└──────────┘           └──────────────┘
                             │
                        检测电机磁场
```

### 测速原理

```
电机每转一圈 → 霍尔传感器输出 N 个脉冲（N = 极对数 × 2）

RPM = (脉冲计数 / 极对数) × (60 / 采样时间)

本项目极对数 = 7（config.h 中 HALL_POLE_PAIRS）
```

## ⚙️ 代码配置

在 `src/config.h` 中修改引脚配置：

```cpp
// ============================================================================
// ST7735 TFT LCD 引脚配置 (SPI)
// ============================================================================
#define USE_LCD true                        // 启用/禁用 LCD 显示
#define TFT_CS    7                         // 片选
#define TFT_RST   10                        // 复位
#define TFT_DC    6                         // 数据/命令
#define TFT_MOSI  3                         // SPI MOSI (SDA)
#define TFT_SCLK  2                         // SPI 时钟 (SCK)

// ============================================================================
// 继电器引脚配置
// ============================================================================
#define RELAY_PIN 8                         // 继电器控制引脚
#define RELAY_ACTIVE_LOW true               // 低电平触发

// ============================================================================
// ESC PWM 配置
// ============================================================================
#define ESC_PWM_PIN 1                       // ESC 信号线 GPIO
#define ESC_PWM_FREQ 50                     // PWM 频率 (Hz)
#define ESC_MIN_PULSE 1000                  // 最小脉宽 (us)
#define ESC_MAX_PULSE 2000                  // 最大脉宽 (us)

// ============================================================================
// INA219 电流传感器引脚 (I2C)
// ============================================================================
#define INA219_SDA 4                        // I2C 数据
#define INA219_SCL 5                        // I2C 时钟

// ============================================================================
// 霍尔传感器引脚
// ============================================================================
#define HALL_PIN 9                          // 霍尔传感器中断引脚
#define HALL_POLE_PAIRS 7                   // 电机极对数
```

## ❓ 常见问题

### Q: LCD 显示花屏或闪烁？

检查 SPI 接线是否牢固（SCK, SDA, CS），确保 VCC 供电充足。

### Q: LCD 完全不显示？

1. 检查 VCC/GND 是否正确连接
2. 尝试使用 5V 供电
3. 检查背光引脚是否连接

### Q: 继电器不动作？

1. 检查 VCC/GND 接线
2. 确认触发电平（高/低电平触发）
3. 用万用表测量 IN 引脚电平变化

### Q: ESC 无响应？

1. 检查 PWM 信号线连接
2. 确认 PWM 频率为 50Hz
3. 尝试重新校准 ESC

### Q: INA219 读数为 0？

1. 检查 I2C 接线（SDA, SCL）
2. 确认 I2C 地址（默认 0x40）
3. 检查 VIN+/VIN- 是否串联在电路中

### Q: 霍尔传感器无转速读数？

1. 检查传感器位置是否靠近电机磁场
2. 确认极对数配置正确
3. 用示波器检查输出脉冲

## ✅ 接线检查清单

上传代码前，请确认：

- [ ] 所有 GND 已共地连接
- [ ] LCD VCC 使用 5V 供电
- [ ] 继电器触发电平配置正确
- [ ] ESC 信号线连接到 GPIO 1
- [ ] INA219 串联在电源回路中
- [ ] 霍尔传感器靠近电机磁场
- [ ] 没有引脚冲突（同一 GPIO 连接多个设备）
