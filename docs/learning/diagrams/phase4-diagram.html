⚠️ 积分饱和问题

❌ 无抗饱和

电机堵转场景

1. 目标 RPM=3000，实际=0
2. 误差=3000，积分项不断累积
3. 输出达到 100% 饱和
4. 堵转解除后，积分仍很大
5. 电机狂飙失控！

✅ 有抗饱和

限幅保护

1. 目标 RPM=3000，实际=0
2. 误差=3000，积分累积
3. 积分值被限制在上限
4. 堵转解除后，积分正常
5. 平稳恢复正常控制
integralMax = (outputMax - P) / Ki

当 P + I 接近输出上限时，自动限制积分值

RPM 计算公式

RPM = (Δp × 60) / (N × Δt)

Δp = 脉冲增量
N = 极对数 (本项目: 7)
Δt = 时间间隔 (秒)

🎛️ PID 调参流程

1
关闭 I 和 D，调 Kp
Ki=0, Kd=0，逐步增大 Kp 直到开始震荡，记录 Kc 和 Tc
2
设置初始参数
Kp = 0.6×Kc，Ki = 2×Kp/Tc，Kd = Kp×Tc/8
3
微调优化
超调大 → 增 Kd、减 Kp | 响应慢 → 增 Kp | 稳态误差 → 增 Ki

💡 调参顺序：Kp（响应）→ Kd（阻尼）→ Ki（精度）
