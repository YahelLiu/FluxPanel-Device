⚖️ 配置存储方式对比

#define 宏定义

#define ESC_FREQ 50
  • ✓ 存储于 Flash
  • ✓ 编译期替换，最快
  • ✓ 不占用 RAM
  • ✗ 修改需重新编译
  • ✗ 无类型检查

推荐：不变配置

const 常量

const int freq = 50;
  • ✓ 存储于 Flash
  • ✓ 有类型检查
  • ✓ 可用于调试
  • ✗ 占用 RAM（某些情况）
  • ✗ 运行时常量

适用：只读参数

变量

int freq = 50;
  • ✓ 运行时可修改
  • ✓ 可动态调整
  • ✓ 有类型检查
  • ✗ 占用 RAM
  • ✗ 访问较慢

适用：可调参数

💡 为什么需要中间状态？ ESC 解锁需要时间，电机有惯性，防止命令冲突

🛡️ Fail-safe 安全机制

硬件层
继电器可控电源
GPIO 默认低电平
保险丝保护
通信层
WebSocket 断开 → 停机
命令超时 10s → 停机
WiFi 断开 → 停机
状态机层
未校准禁止控制
状态转换条件明确
错误状态自动停机
保护层
过流保护 >5A
过温保护 >80°C
堵转保护 3秒
