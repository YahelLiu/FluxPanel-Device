# FluxPanel Device - 开发计划

## 📋 项目概述

FluxPanel 系统的 ESP32-C3 设备端，实现：
- 远程监控 PC 状态（CPU/GPU/MEM）
- 远程硬件控制（电机/继电器等）
- LCD 实时显示系统状态
- WebSocket 与 FluxPanel 后端通信

## 🔧 详细任务分解

### ESC PWM 控制 (P0)

**目标**: 通过 PWM 信号控制 BLDC 电机转速

**技术方案**:
```
ESP32 GPIO ──→ 50Hz PWM ──→ ESC 信号线 ──→ BLDC 电机

PWM 参数:
- 频率: 50Hz
- 脉宽范围: 1ms (停止) ~ 2ms (全速)
- 分辨率: 16-bit
```

**任务清单**:
- [ ] 选择 ESC PWM 信号 GPIO
- [ ] 实现 PWM 初始化 (ledc)
- [ ] 实现转速 0-100% 映射到脉宽
- [ ] 添加 ESC 启动校准流程
- [ ] 集成到命令处理逻辑
- [ ] 更新 LCD 显示转速

**预计工作量**: 2-3 小时

---

### INA219 电流传感器 (P1)

**目标**: 实时监测电机电流和功率

**技术方案**:
```
INA219 ──→ I2C (GPIO4/GPIO5) ──→ ESP32

测量数据:
- 电流 (A)
- 电压 (V)
- 功率 (W)
```

**任务清单**:
- [ ] 添加 INA219 库依赖
- [ ] 初始化 I2C 和 INA219
- [ ] 校准电流测量范围
- [ ] 定时读取传感器数据
- [ ] 上报电流数据到服务器
- [ ] LCD 显示电流/功率

**预计工作量**: 2-4 小时

---

### 霍尔传感器测速 (P1)

**目标**: 测量电机实际转速

**技术方案**:
```
霍尔传感器 ──→ GPIO9 (中断) ──→ ESP32

测量方式:
- 计数脉冲数量
- 计算转速 (RPM)
```

**任务清单**:
- [ ] 配置 GPIO 中断
- [ ] 实现脉冲计数
- [ ] 计算 RPM
- [ ] 添加滤波算法
- [ ] 上报转速数据
- [ ] LCD 显示实际转速

**预计工作量**: 2-3 小时

---

### 报警逻辑 (P1)

**目标**: 异常情况自动保护

**报警条件**:
| 条件 | 阈值 | 动作 |
|------|------|------|
| 过流 | > 5A | 立即停止 + 继电器断开 |
| 过温 | > 80°C | 降低转速 + 警告 |
| 堵转 | RPM=0 且 PWM>0 | 立即停止 |
| 通信断开 | WS 断开 > 30s | 安全停车 |

**任务清单**:
- [ ] 定义报警阈值配置
- [ ] 实现过流检测
- [ ] 实现过温检测
- [ ] 实现堵转检测
- [ ] 实现通信超时检测
- [ ] 报警状态上报
- [ ] LCD 报警显示

**预计工作量**: 3-4 小时
